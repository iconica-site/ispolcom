  2. Цели обработки персональных данных
    1. Сайт собирает и хранит только ту персональную информацию, которая необходима для предоставления сервисов или исполнения соглашений и договоров с Посетителем, за исключением случаев, когда законодательством предусмотрено обязательное хранение персональной информации в течение определенного законом срока.
    2. Оператор обрабатывает персональные данные для осуществления деятельности, реализации своих законных интересов и требований.
    3. Цели обработки персональных данных:
      • предоставление Посетителю доступа к ресурсам Сайта;
      • предоставление Посетителям Сайта консультаций;
      • установление с Посетителем обратной связи, включая направление уведомлений, запросов, касающихся использования Сайта, оказания услуг, обработку запросов и заявок от Посетителя;
      • определение места нахождения Посетителя для обеспечения безопасности, предотвращения мошенничества.
  3. Порядок и условия обработки, хранения персональных данных
    1. В отношении персональных данных Посетителя сохраняется конфиденциальность, кроме случаев добровольного предоставления Посетителем информации о себе для общего доступа неограниченному кругу лиц. В данном случае Посетитель соглашается с тем, что определенная часть его персональной информации становится общедоступной.
    2. Оператор вправе передать персональную информацию Посетителя третьим лицам в следующих случаях:
      • передачи персональных данных уполномоченным государственным органам в пределах их полномочий в соответствии с российским законодательством или иным применимым законодательством;
      • когда передача персональных данных осуществляется в целях исполнения договора с субъектом персональных данных;
      • передачи персональных данных правопреемнику ИСПОЛКОМ при его реорганизации, а также лицам, которым ИСПОЛКОМ поручает обработку персональных данных;
      • в иных случаях, когда Посетитель выразил согласие на такие действия.
    3. При обработке персональных данных Оператор вправе осуществлять сбор, систематизацию, накопление, хранение, уточнение (обновление, изменение), извлечение, использование, передачу, удаление, блокирование, уничтожение персональных данных Посетителя, в том числе в информационных системах персональных данных с использованием средств автоматизации или без использования таких средств.
    4. Обработка персональных данных Пользователя осуществляется без ограничения срока любым законным способом.
    5. Персональные данные хранятся в течение срока, необходимого для целей, в которых такие данные были предоставлены или в течение периода, установленного законодательством, договором, соглашением с субъектом персональных данных, сроком исковой давности.
    6. По достижении целей, либо истечения периода обработки персональных данных они уничтожаются.
  4. Обязательства и обязанности сторон
    1. Субъект персональных данных вправе:
      • получать перечень своих персональных данных, обрабатываемых Оператором, источник их получения, цели и применяемые Оператором способы обработки персональных данных, сроки обработки персональных данных, в том числе сроки хранения, иные сведения, предусмотренные действующим законодательством Российской Федерации;
      • требовать уточнения своих персональных данных, их блокирования или уничтожения;
      • обжаловать действия или бездействия Оператора в установленном законом порядке.
    2. Посетитель обязан:
      • предоставлять только достоверные данные о себе;
      • обновлять, уточнять предоставленную информацию о себе в случае изменения данной информации.
    3. Оператор обязан:
      • использовать полученную информацию исключительно в пределах заявленных целей;
      • обеспечить хранение конфиденциальной информации в тайне. Не сообщать без предварительного письменного разрешения Посетителя его персональные данные в коммерческих целях;
      • проводить мероприятия по защите персональных данных Посетителей Сайта в соответствии с настоящей Политикой, иными документами, изданными в ИСПОЛКОМ.
  5. Реализация защиты персональных данных
    1. Деятельность Оператора по обработке персональных данных в информационных системах основана на принципах защиты конфиденциальности полученной информации.
    2. К мерам, применяемым Оператором, для защиты персональных данных относятся:
      • назначение ответственного лица за организацию обработки персональных данных;
      • применение правовых, технических и организационных мер по обеспечению защиты персональных данных;
      • использование охраняемых помещений с разграниченным доступом для размещения серверов информационных систем с персональными данными;
      • проведение мероприятий, направленных на предотвращение несанкционированного доступа к персональным данным;
      • ознакомление сотрудников Оператора, непосредственно осуществляющих обработку персональных данных, с требованиями законодательства Российской Федерации о персональных данных;
      • своевременное обнаружение фактов несанкционированного доступа к персональным данным
      • возможность незамедлительного восстановления персональных данных, модифицированных и уничтоженных вследствие несанкционированного доступа к ним;
      • постоянный контроль уровня защищенности персональных данных.
    3. ИСПОЛКОМ добросовестно относится к вопросу защиты собранных персональных данных реализует требования к их защите путем принятия комплекса мер, необходимых и достаточных для обеспечения выполнения обязанностей, предусмотренных действующим законодательством Российском Федерации о персональных данных.
    4. Оператор не несет ответственности за получение третьими лицами персональных данных путем получения доступа к личному кабинету зарегистрированных пользователей. Зарегистрированные пользователи Сайта обязаны самостоятельно обеспечивать сохранность данных для доступа к своему личному кабинету.
